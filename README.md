# Python Trivia Game 🎮

A command-line Python Trivia Quiz Game with multiple difficulty levels, scoring, streak bonuses, lives, leaderboard tracking, automated testing using Pytest, and Continuous Integration using GitHub Actions.

## Features

* Easy / Medium / Hard difficulty levels
* Random question selection
* Hint system
* Score tracking
* Streak bonus scoring
* Lives system
* Leaderboard
* Automated testing with Pytest
* Continuous Integration using GitHub Actions

## Installation

Clone repository:

```bash
git clone <your-repository-url>
```

Move into project:

```bash
cd "Trivia Game"
```

Install dependencies:

```bash
pip install pytest
```

---

## Run the Game

```bash
python project1.py
```

---

## Run Tests

```bash
pytest
```

---

## Continuous Integration

GitHub Actions automatically runs tests whenever code is pushed to the repository.
